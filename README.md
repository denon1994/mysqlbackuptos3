## Mysql DB backup to S3 bucket in AWS
A bash script to get MySql backup and push it to S3 bucket. <br />
Change below parameters before use. <br />
Set your database credentials
DB_USER="" <br />
DB_PASS="" <br />
DB_NAME=""

Set AWS credentials <br />
AWS_ACCESS_KEY="" <br />
AWS_SECRET_KEY="" <br />
S3_BUCKET=""
